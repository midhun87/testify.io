AI Developer

AUTOMATION Remote

We are seeking a creative and skilled AI Developer to build intelligent systems and AI-powered applications for our platform. You will design, implement, and deploy machine learning and AI solutions that solve real-world problems.

Responsibilities:

  • Design and develop AI/ML models for NLP, computer vision, or predictive analytics.
  • Build APIs and services that integrate AI capabilities into applications.

  • Strong experience with Python and ML libraries (TensorFlow, PyTorch, Scikit-learn).
  • Understanding of ML algorithms, neural networks, and deep learning.
  • Experience in RESTful API development and MLOps.

Compiler API Developer

ENGINEERING Remote

We are looking for a passionate and detail-oriented Compiler API Developer to join our engineering team. The candidate will be responsible for designing, building, and maintaining compiler APIs, language tooling, and related systems.

Responsibilities:

  • Design and develop compiler APIs, parsers, and language runtimes.
  • Optimize code generation, execution speed, and memory management.
  • Debug and resolve compiler-related issues and performance bottlenecks.
  • Write clean, scalable, and well-documented code.

Requirements:

  • Strong proficiency in C/C++, Rust, or Java.
  • Hands-on experience with compiler theory (lexers, parsers, ASTs, etc.).
  • Knowledge of LLVM, GCC, or similar compiler frameworks.
